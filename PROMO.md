# Deep Search Promo Kit

## One-Line Pitch

Deep Search is a local-first research skill that turns a vague question into an exhaustive, multi-source investigation workflow.

## Short Description

Deep Search packages a serious research workflow into a reusable local skill. It combines broad web coverage, news aggregation, academic lookup, and structured synthesis so your agent can investigate a topic instead of just summarizing the first few results.
